# Project Documentation

This directory contains detailed documentation for specific components of the `VASP2WAN90_v2_fix` project.

## Available Documents

*   **[`mlwf_patch.md`](./mlwf_patch.md)**:
    Detailed documentation for the `mlwf.patch` file. This patch modifies VASP's `VASP2WANNIER90v2` interface, providing enhanced control over output files and improved capabilities for non-collinear Wannier function calculations and spinor projections.

*   **[`w90unk2unk_f90.md`](./w90unk2unk_f90.md)**:
    Documentation for the `utility/w90unk2unk/w90unk2unk.f90` Fortran utility. This program converts Wannier90 `UNK` files (which store Bloch wavefunctions) between their binary (unformatted) and text (formatted) representations.

Please refer to the individual files for specific details on each component.
